## Run Locally

Clone the project

```bash
  git clone https://github.com/pyroTM/nyx
```

Go to the project directory

```bash
  cd nyx-master
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Edit Configuration Values

```bash
  token = 'your token here'
  prefix = 'your prefix here'
```

Start Nyx

```bash
  py nyx.py
```
